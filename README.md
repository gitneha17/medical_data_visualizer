# Medical Data Visualizer

## 📌 Description

A Python project that analyzes and visualizes medical examination data using Pandas, Matplotlib, and Seaborn. The project explores relationships between cardiovascular disease, body measurements, lifestyle habits, and medical indicators through categorical plots and correlation heatmaps.

This project was completed as part of the **freeCodeCamp Data Analysis with Python Certification**.


## 🎯 Objective

The objective of this project is to practice:

* Data cleaning and preprocessing
* Feature engineering
* Data visualization with Seaborn and Matplotlib
* Correlation analysis
* Exploratory Data Analysis (EDA)

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 🚀 Features

* Calculates BMI and creates an **overweight** feature.
* Normalizes cholesterol and glucose values.
* Generates a **categorical plot** to compare health indicators.
* Creates a **correlation heatmap** to identify relationships between variables.
* Performs data cleaning by removing incorrect blood pressure and outlier values.
